INSTRUÇÕES PARA COMEÇAR:

1. Crie um projeto no Firebase: https://console.firebase.google.com/
2. Ative a autenticação com Google e crie um Firestore Database.
3. Copie as credenciais do Firebase (apiKey, authDomain, etc.) e cole no main.js
4. Publique este site no https://netlify.com ou https://vercel.com para acessá-lo de qualquer lugar.
5. Adicione funcionalidades conforme desejar. Esta é a base inicial com login e estrutura de anotações.
